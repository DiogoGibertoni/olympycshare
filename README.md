# 🏅 OlympycShare

Rede social esportiva para conexão e organização de eventos.

## Sobre o Projeto

O OlympycShare conecta pessoas que desejam praticar esportes, permitindo a criação, descoberta e participação em eventos esportivos.

## Tech Stack

- **Frontend:** Next.js 15, React 19, TypeScript
- **Backend:** Next.js API Routes
- **Banco de dados:** A definir (MongoDB ou PostgreSQL)
- **Autenticação:** JWT

## Como Rodar

``````bash
npm install
npm run dev
# http://localhost:3000
``````

- **Método:** Scrum adaptado com Kanban
